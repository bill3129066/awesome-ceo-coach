# Reference Index — Awesome CEO Coach

Each distilled reference contains the actionable frameworks. The original sources are listed for further reading.

## How to Use This Index

When coaching, find the relevant topic below. Each row maps to:
1. **Distilled Reference** — actionable SOPs, frameworks, dos/don'ts (read this first)
2. **Mochary Chapters** — chapter references from *The Great CEO Within*
3. **Bailey Articles** — articles at dave-bailey.com for additional reading
